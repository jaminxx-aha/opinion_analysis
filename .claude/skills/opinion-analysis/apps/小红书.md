# 小红书

小红书是一款生活方式分享平台。

## 内容模块

**页面结构：**
- 首页推荐页 - 笔记推荐流
- 笔记详情页 - 单篇笔记内容
- 搜索结果页 - 笔记/商品搜索

**常见场景：**
- 浏览笔记 - 查看推荐内容
- 查看笔记详情 - 点开单篇笔记
- 搜索内容 - 搜索笔记或商品

## 发布模块

**页面结构：**
- 笔记发布页 - 发布图文/视频笔记

**常见场景：**
- 发布笔记 - 发布图文或视频内容
- 编辑笔记 - 修改笔记内容

## 商城模块

**页面结构：**
- 商城首页 - 商品推荐
- 商品详情页 - 商品信息
- 购物车页 - 待购买商品

**常见场景：**
- 商品浏览 - 查看商品推荐
- 商品购买 - 下单购买商品
