# 小红书

小红书是一款生活方式分享平台。

## 内容模块

**页面结构：**
- 首页推荐页 - 笔记推荐流
- 笔记详情页 - 单篇笔记内容
- 搜索结果页 - 笔记/商品搜索

**常见场景：**
- 浏览笔记 - 查看推荐内容
- 查看笔记详情 - 点开单篇笔记
- 搜索内容 - 搜索笔记或商品
- 笔记互动 - 点赞、收藏笔记
- 关注创作者 - 关注感兴趣的作者

## 发布模块

**页面结构：**
- 笔记发布页 - 发布图文/视频笔记

**常见场景：**
- 发布笔记 - 发布图文或视频内容
- 编辑笔记 - 修改笔记内容
- 选择滤镜 - 应用图片滤镜
- 添加标签 - 选择话题标签

## 商城模块

**页面结构：**
- 商城首页 - 商品推荐
- 商品详情页 - 商品信息
- 购物车页 - 待购买商品

**常见场景：**
- 商品浏览 - 查看商品推荐
- 商品购买 - 下单购买商品
- 商品评价 - 对已购商品评价
- 好物分享 - 分享购买体验

## 互动模块

**页面结构：**
- 评论页 - 笔记评论区
- 收藏页 - 个人收藏列表
- 关注页 - 关注创作者列表

**常见场景：**
- 发送评论 - 对笔记发表评论
- 收藏笔记 - 收藏感兴趣的内容
- 分享笔记 - 分享给好友或朋友圈
