# 微信

微信是一款综合社交通讯应用。

## 聊天模块

**页面结构：**
- 聊天列表 - 所有对话列表
- 聊天对话页 - 单聊/群聊界面
- 聊天详情页 - 聊天设置、成员管理

**常见场景：**
- 发送消息 - 文字、语音、图片、视频
- 接收消息 - 查看新消息
- 群聊管理 - 创建群、管理群成员
- 消息转发 - 转发消息给其他联系人
- 语音通话 - 单聊/群聊语音通话
- 视频通话 - 单聊视频通话

## 朋友圈模块

**页面结构：**
- 朋友圈首页 - 好友动态列表
- 个人朋友圈页 - 自己的动态
- 发布页 - 发布动态界面

**常见场景：**
- 浏览朋友圈 - 查看好友动态
- 发布动态 - 发布图文/视频
- 点赞评论 - 互动操作
- 朋友圈广告 - 广告内容展示
- 朋友圈相册 - 查看历史动态

## 视频号模块

**页面结构：**
- 视频号首页 - 推荐视频列表
- 视频播放页 - 播放单个视频
- 视频号主页 - 创作者主页

**常见场景：**
- 观看视频 - 浏览视频号内容
- 视频互动 - 点赞、评论视频号内容
- 视频号直播 - 观看直播内容

## 小程序模块

**页面结构：**
- 小程序列表 - 已使用小程序
- 小程序页面 - 各小程序内部页面

**常见场景：**
- 打开小程序 - 通过分享链接或搜索进入
- 小程序购物 - 在小程序内下单购买
- 小程序游戏 - 在小程序内玩游戏
- 小程序支付 - 在小程序内完成支付

## 支付模块

**页面结构：**
- 收付款页 - 扫码支付、收款
- 钱包页 - 余额、银行卡管理
- 理财页 - 理财通产品

**常见场景：**
- 扫码支付 - 扫二维码付款
- 转账 - 给好友转账
- 收款 - 生成收款二维码
- 红包 - 发红包、领红包
- 理财 - 购买理财产品
