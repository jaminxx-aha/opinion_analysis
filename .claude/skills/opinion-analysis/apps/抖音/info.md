# 抖音

抖音是一个综合性短视频社交平台，包含以下主要功能模块：

## 短视频模块

核心功能，用户浏览、发布短视频内容。

**页面结构：**
- 首页推荐页 - 系统推荐的短视频流
- 关注页 - 关注创作者的视频流
- 视频播放页 - 单个视频的播放界面
- 视频发布页 - 录制/上传视频的界面
- 视频详情页 - 包含评论、点赞、分享等互动功能

**常见场景：**
- 视频播放 - 观看视频内容
- 滑动切换 - 上下滑动切换视频
- 视频互动 - 点赞、评论、分享
- 视频发布 - 录制、编辑、上传

## 直播模块

实时直播功能，创作者与观众实时互动。

**页面结构：**
- 直播广场 - 直播推荐列表
- 直播播放页 - 观看直播界面
- 直播互动区 - 发送弹幕、点赞、送礼
- 直播购物车 - 直播带货商品列表

**常见场景：**
- 直播观看 - 观看直播内容
- 弹幕互动 - 发送弹幕、点赞
- 直播购物 - 点击购物车、购买商品
- 直播开播 - 开启直播

## 商城模块

电商购物功能，短视频+直播带货。

**页面结构：**
- 商城首页 - 商品推荐、分类入口
- 商品详情页 - 商品信息、购买按钮
- 搜索结果页 - 商品搜索结果列表
- 购物车页 - 已添加商品列表
- 订单页 - 订单详情、物流跟踪

**常见场景：**
- 商品浏览 - 查看商品列表
- 商品搜索 - 搜索特定商品
- 商品购买 - 下单、支付
- 订单管理 - 查看订单、退换货

## 消息模块

社交互动功能。

**页面结构：**
- 消息列表 - 私信、通知、粉丝消息
- 聊天页 - 与用户私信对话
- 通知页 - 系统通知、活动通知

## 个人中心

用户个人信息管理。

**页面结构：**
- 个人主页 - 用户作品、喜欢、收藏
- 设置页 - 账号设置、隐私设置
- 钱包页 - 抖音币、提现管理
