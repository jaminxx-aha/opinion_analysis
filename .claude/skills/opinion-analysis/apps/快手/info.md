# 快手

快手是一款短视频社交平台，结构与抖音类似。

## 短视频模块

**页面结构：**
- 首页推荐页 - 系统推荐的短视频流
- 关注页 - 关注创作者的视频流
- 视频播放页 - 单个视频的播放界面
- 视频发布页 - 录制/上传视频的界面

**常见场景：**
- 视频播放 - 观看视频内容
- 滑动切换 - 上下滑动切换视频
- 视频互动 - 点赞、评论、分享
- 视频收藏 - 收藏喜欢的视频

## 直播模块

**页面结构：**
- 直播广场 - 直播推荐列表
- 直播播放页 - 观看直播界面
- 直播互动区 - 发送弹幕、点赞、送礼

**常见场景：**
- 观看直播 - 进入直播间观看
- 直播互动 - 发弹幕、送礼物
- 直播购物 - 在直播间购买商品
- 直播连麦 - 主播与观众连麦互动

## 商城模块

**页面结构：**
- 商城首页 - 商品推荐
- 商品详情页 - 商品信息
- 购物车页 - 待购买商品
- 订单页 - 订单详情

**常见场景：**
- 浏览商品 - 查看推荐商品
- 商品下单 - 购买商品
- 查看订单 - 查看购买记录
- 商品评价 - 对已购商品评价
