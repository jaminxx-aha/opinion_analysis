## 分类歧义说明

部分分类项在不同一级分类下含义不同，判断时需注意区分：

| 表面描述 | 分类路径 A | 分类路径 B | 判断依据 |
|---------|-----------|-----------|---------|
| 直播画面卡顿 | 卡顿.直播卡顿.直播观看画面卡顿（UI渲染层不流畅） | 网络异常.直播网络异常.直播画面卡顿（网络层缓冲/断流） | "缓冲/加载/断流/重连" → 网络异常；"卡帧/一顿一顿/不流畅" → 卡顿 |
| 视频加载不出 | 卡顿.加载卡顿.视频封面加载卡顿（本地渲染慢） | 网络异常.视频网络异常.视频加载失败（网络请求失败） | "网络/断网/超时" → 网络异常；"刷新/卡住" → 卡顿 |
| 视频画面模糊 | 渲染异常.视频渲染异常.视频画面模糊（解码/渲染问题） | 网络异常.视频网络异常.视频画质自动降低（网络自适应降画质） | "自适应/降低/网速慢" → 网络异常；"花屏/色差/始终模糊" → 渲染异常 |

## 其他应用分类适配

一级分类（卡顿、响应慢/延迟、闪退/崩溃、启动异常、发热、内存异常、渲染异常、网络异常）对所有应用通用。二级和三级分类需根据应用的功能模块做适配。以下列出各非抖音应用与抖音分类的关键差异。

### 微信

微信没有视频拍摄/剪辑、直播购物车等功能模块，二级分类需替换为微信特有场景：

| 一级分类 | 抖音二级分类 | 微信适配二级分类 | 三级分类示例 |
|---------|------------|---------------|-----------|
| 卡顿 | 滑动卡顿 | 聊天滑动卡顿 | 聊天列表滑动卡顿、朋友圈滑动卡顿 |
| 卡顿 | 视频播放卡顿 | 视频号播放卡顿 | 视频号首页播放卡顿、视频号详情播放卡顿 |
| 卡顿 | 直播卡顿 | 视频号直播卡顿 | 视频号直播画面卡顿 |
| 卡顿 | 页面转场卡顿 | 聊天转场卡顿 | 聊天列表进入对话页转场卡顿、朋友圈进入详情转场卡顿 |
| 卡顿 | 拍摄/剪辑卡顿 | (无此模块) | -- |
| 卡顿 | 互动操作卡顿 | 聊天互动卡顿 | 消息发送卡顿、朋友圈点赞卡顿 |
| 卡顿 | 加载卡顿 | 聊天加载卡顿 | 聊天列表加载卡顿、朋友圈加载卡顿 |
| 网络异常 | 交互网络异常 | 聊天网络异常 | 消息发送失败、朋友圈上传失败 |
| 网络异常 | 商城网络异常 | 支付网络异常 | 微信支付失败、转账失败 |
| 网络异常 | 视频网络异常 | 视频号网络异常 | 视频号加载失败 |
| 渲染异常 | 视频渲染异常 | 视频号渲染异常 | 视频号画面模糊 |

淘宝以购物流程为核心，没有短视频播放、直播观看等功能（淘宝直播属电商直播）：

| 一级分类 | 抖音二级分类 | 淘宝适配二级分类 | 三级分类示例 |
|---------|------------|---------------|-----------|
| 卡顿 | 滑动卡顿 | 商品列表滑动卡顿 | 首页推荐商品滑动卡顿、搜索结果滑动卡顿 |
| 卡顿 | 视频播放卡顿 | (无此模块) | -- |
| 卡顿 | 直播卡顿 | 直播间观看卡顿 | 淘宝直播间画面卡顿 |
| 卡顿 | 页面转场卡顿 | 购物流程转场卡顿 | 首页进入商品详情转场卡顿、商品详情进入购物车转场卡顿 |
| 卡顿 | 拍摄/剪辑卡顿 | (无此模块) | -- |
| 卡顿 | 互动操作卡顿 | 购物互动卡顿 | 商品收藏卡顿、评价提交卡顿 |
| 卡顿 | 加载卡顿 | 商品加载卡顿 | 首页加载卡顿、商品图片加载卡顿 |
| 网络异常 | 交互网络异常 | 购物交互网络异常 | 搜索失败、评价提交失败 |
| 网络异常 | 商城网络异常 | (通用) | 下单失败、支付失败、订单查询失败 |
| 网络异常 | 视频网络异常 | (无此模块) | -- |

## 分类使用说明

### 1. 分类选择原则

- **从大到小**：先确定一级大类，再细分二级、三级
- **场景匹配**：三级分类必须对应具体应用场景
- **优先核心问题**：当问题涉及多个分类时，优先选择核心问题类型
- **歧义项注意**：对于歧义说明中列出的项，根据用户描述中的关键词判断归属

### 2. 典型问题分类示例

| 用户反馈 | 分类结果 |
|---------|---------|
| "抖音刷视频卡顿严重" | 卡顿.滑动卡顿.首页推荐视频流上下滑动卡顿 |
| "抖音看直播画面卡顿" | 卡顿.直播卡顿.直播观看画面卡顿 |
| "抖音视频播放一直卡住不动" | 卡顿.视频播放卡顿.首页推荐视频播放卡顿 |
| "抖音上下滑动切换视频卡顿" | 卡顿.滑动卡顿.首页推荐视频流上下滑动卡顿 |
| "抖音打开很慢" | 启动异常.启动慢.冷启动慢 |
| "抖音看视频发热严重" | 发热.视频播放发热.长时间播放发热 |
| "抖音直播画面模糊" | 渲染异常.直播渲染异常.直播画面模糊 |
| "抖音视频加载不出来" | 网络异常.视频网络异常.视频加载失败 |
| "抖音发布视频卡顿" | 卡顿.拍摄/剪辑卡顿.视频发布上传卡顿 |
| "抖音切到评论区卡顿" | 卡顿.页面转场卡顿.视频页进入评论区转场卡顿 |

### 3. 扩展分类

当遇到新场景不在当前分类体系中时，可按以下规则扩展：

1. 保持一级、二级分类不变
2. 在三级分类中新增具体场景
3. 新增分类需遵循命名规范：`{页面/功能}.{具体操作}.{问题表现}`
