# 快手

快手是一款短视频社交平台，结构与抖音类似。

## 短视频模块

**页面结构：**
- 首页推荐页 - 系统推荐的短视频流
- 关注页 - 关注创作者的视频流
- 视频播放页 - 单个视频的播放界面
- 视频发布页 - 录制/上传视频的界面

**常见场景：**
- 视频播放 - 观看视频内容
- 滑动切换 - 上下滑动切换视频
- 视频互动 - 点赞、评论、分享

## 直播模块

**页面结构：**
- 直播广场 - 直播推荐列表
- 直播播放页 - 观看直播界面
- 直播互动区 - 发送弹幕、点赞、送礼

## 商城模块

**页面结构：**
- 商城首页 - 商品推荐
- 商品详情页 - 商品信息
- 购物车页 - 待购买商品
- 订单页 - 订单详情
